# AGENTS.md - Codex（ミコト）向け作業規約

## リポジトリ構成と編集対象

- **HTMLツール類の編集対象は常に `_publish_soltina/` 配下**です。
  例：`_publish_soltina/speed-calc.html`
- 作業ルート直下に同名のHTMLファイルが存在する場合がありますが、
  **Git管理外のローカルコピー**です。編集・同期の対象にしないでください。
- 指示書に記載されたパスと実ファイルの場所が食い違う場合は、
  作業を進めず、その旨を報告してください。

## コミット規約

- 1タスク＝1コミット。無関係な未コミット差分を混ぜないこと。
- `git add` はファイル指定で行うこと（`git add -A` 禁止）。
- コミットメッセージは `fix(ツール名): 修正内容の要約` の形式。

## 作業完了時の報告事項

- 変更したファイルのパス一覧
- コミットハッシュ（コミットした場合）
- 指示書の検証項目に対する確認結果
- 指示外で気づいた問題（あれば。勝手に修正しないこと）

## 禁止事項

- 指示書にない機能の追加・変更（IDEAS.md記載の未依頼機能を含む）
- `_publish_soltina/` 以外へのHTMLツールの新規作成
- 無関係ファイルの整形・リネーム・削除
