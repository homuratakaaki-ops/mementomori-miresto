# AGENTS.md - Codex（ミコト）向け作業規約

## リポジトリ構成と編集対象

- **HTMLツール類の編集対象はリポジトリルート直下**です。
  例：`speed-calc.html`
- リポジトリルート直下のHTMLファイルは、Git管理下の本番ファイルです。
- 共有データは `data/` 配下で管理します。
- 指示書に記載されたパスと実ファイルの場所が食い違う場合は、
  作業を進めず、その旨を報告してください。

## コミット規約

- 1タスク＝1コミット。無関係な未コミット差分を混ぜないこと。
- `git add` はファイル指定で行うこと（`git add -A` 禁止）。
- コミットメッセージは `fix(ツール名): 修正内容の要約` の形式。

## 作業完了時の報告事項

- 変更したファイルのパス一覧
- コミットハッシュ（コミットした場合）
- 指示書の検証項目に対する確認結果
- 指示外で気づいた問題（あれば。勝手に修正しないこと）

## 禁止事項

- 指示書にない機能の追加・変更（IDEAS.md記載の未依頼機能を含む）
- 無関係ファイルの整形・リネーム・削除
